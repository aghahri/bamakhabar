# راهنمای مایگریشن برای قابلیت‌های جدید

بعد از پول کردن این تغییرات، این کارها را انجام دهید:

## ۱. مایگریشن دیتابیس

فیلد `description` به مدل Neighborhood و مدل جدید `NewsletterSubscription` اضافه شده است.

```bash
npx prisma migrate dev --name add_description_and_newsletter
```

در محیط production:

```bash
npx prisma migrate deploy
```

سپس در صورت نیاز:

```bash
npx prisma generate
```

---

## ۲. قابلیت‌های اضافه‌شده

- **جستجو:** صفحه `/search` و لینک در هدر — جستجو در عنوان/خلاصه/متن خبر + فیلتر استان و شهر
- **فید RSS:** `/rss` برای همه اخبار، `/rss/[provinceSlug]` برای اخبار هر استان
- **نشان منطقه روی کارت خبر:** در لیست اخبار نمایش «محله، شهر» روی هر کارت
- **درباره این محله:** در ادمین → محلات می‌توانید برای هر محله توضیح (description) وارد کنید؛ در صفحه همان محله نمایش داده می‌شود
- **آمار ادمین:** صفحه `/admin/stats` — تعداد اخبار، محلات پرخبر، نمودار ساده
- **خبرنامه:** فرم در فوتر — کاربران با ایمیل و انتخاب محلات ثبت‌نام می‌کنند. ارسال خودکار ایمیل هنگام انتشار خبر نیاز به تنظیم سرویس ایمیل (مثل Resend) دارد و در نسخهٔ بعدی قابل اضافه شدن است.
