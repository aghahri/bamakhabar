# ایمپورت محلات

فایل zip محلات (`ALLMAHLLAT.zip`) را آنزیپ کنید و تمام فایل‌های `.xlsx` را داخل این پوشه قرار دهید:

```
data/mahallat/
  ├── اسامی محلات استان تهران.xlsx
  ├── نهایی استان قم.xlsx
  └── ...
```

سپس در ریشهٔ پروژه اجرا کنید:

```bash
npm install
npx prisma generate
npm run db:import-mahallat
```

اسکریپت از نام فایل، استان را استخراج می‌کند (مثلاً «اسامی محلات استان گلستان») و در هر شیت به دنبال ستون‌های **استان**، **شهر**/شهرستان، **محله**/نام محله می‌گردد.
